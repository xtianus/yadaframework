# `net.yadaframework.web.datatables.YadaDataTable`

| Method | Description |
|---|---|
| `dtLanguageObj` | Sets the base url where language files are located. |
| `dtAjaxUrl` | Sets the URL where data is fetched from (without language in the path). |
| `dtEntityClass` | Sets the entity class that will be used to fetch data from the database. |
| `dtStructureObj` | Sets the HTML table structure |
| `dtOptionsObj` | Adds the client-side options section. |
