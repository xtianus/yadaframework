# `net.yadaframework.web.datatables.YadaDataTable`

| Method | Description |
|---|---|
| `dtLanguageObj` | Set the base url where language files are located. |
| `dtAjaxUrl` | Set the URL where data is fetched from (without language in the path). |
| `dtEntityClass` | Set the entity class that will be used to fetch data from the database. |
| `dtStructureObj` | Set the HTML table structure |
| `dtOptionsObj` | Adds the client-side options section. |
