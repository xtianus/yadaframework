# `net.yadaframework.web.datatables.options.YadaDTOptions`

| Method | Description |
|---|---|
| `dtData` | Sets the `data` option. |
| `dtDataTableExtErrMode` | Sets the `dataTableExtErrMode` option. |
| `dtAutoWidthOff` | Disable automatic column width calculation as an optimisation. |
| `dtCaption` | Sets the `caption` option. |
| `dtCreatedRow` | Sets the `createdRow` option. |
| `dtDeferLoading` | Enables deferred loading, and instructs DataTables as to how many items are in the full data set |
| `dtDeferRenderOff` | Do not wait to create HTML elements when they are needed for display. |
| `dtDestroy` | Initialise a new DataTable as usual, but if there is an existing DataTable which matches the selector, it will be destroyed and replaced with the new table. |
| `dtTypeDetectOff` | Disable the auto type detection that DataTables performs. |
| `dtDisplayStart` | Sets the `displayStart` option. |
| `dtDrawCallback` | Sets the `drawCallback` option. |
| `dtFooterCallback` | Sets the `footerCallback` option. |
| `dtFormatNumber` | Sets the `formatNumber` option. |
| `dtHeaderCallback` | Sets the `headerCallback` option. |
| `dtInfoOff` | Disables showing details about the table including information about filtered data if that action is being performed. |
| `dtInfoCallback` | Sets the `infoCallback` option. |
| `dtInitComplete` | Sets the `initComplete` option. |
| `dtLayout` | Sets the `layout` option. |
| `dtLengthChangeOff` | Disables user's ability to change the paging display length of the table. |
| `dtLengthMenu` | Sets the `lengthMenu` option. |
| `dtOrderClassesOff` | Disable the addition of ordering classes to the columns when performance is an issue. |
| `dtOrderDescReverseOff` | Disable the default reverse. |
| `dtOrderFixed` | Sets the `orderFixed` option. |
| `dtOrderMultiOff` | Disables multiple column ordering ability |
| `dtOrderingOff` | Disables ordering (sorting) abilities |
| `dtOrder` | Sets the `order` option. |
| `dtPageLength` | Sets the `pageLength` option that tells how many rows should be visible. |
| `dtPagingOff` | Disable table pagination. |
| `dtPreDrawCallback` | Sets the `preDrawCallback` option. |
| `dtProcessingOn` | Enable the display of a 'processing' indicator when the table is being processed (e.g. |
| `dtRenderer` | Sets the `renderer` option. |
| `dtRowCallback` | This callback allows you to 'post process' each row after it have been generated for each table draw, but before it is rendered into the document. |
| `dtRowId` | DataTables will attempt to automatically read an id value from the data source for each row using the property defined by this option. |
| `dtScrollCollapseOn` | Allow the table to reduce in height when a limited number of rows are shown. |
| `dtScrollXOn` | Enable horizontal scrolling. |
| `dtScrollY` | Enable vertical scrolling. |
| `dtSearchDelay` | Sets a delay for search operations. |
| `dtSearchingOff` | Disable searching abilities in DataTables. |
| `dtServerSideOff` | Disable server-side processing. |
| `dtStateDuration` | Duration for which the saved state information is considered valid. |
| `dtStateLoadCallback` | Callback that defines where and how a saved state should be loaded. |
| `dtStateLoadParams` | Callback which allows modification of the saved state prior to loading that state. |
| `dtStateLoaded` | Callback that is fired once the state has been loaded and the saved data manipulated. |
| `dtStateSaveOn` | Enable state saving such as pagination position, display length, filtering and sorting. |
| `dtStateSaveCallback` | Callback that defines how the table state is stored and where. |
| `dtStateSaveParams` | Callback which allows modification of the parameters to be saved for the DataTables state saving prior to the data actually being saved. |
| `dtTabIndex` | Sets the `tabIndex` overrule option. |
| `dtColumnDefsObj` | Sets column defs. |
| `dtColumnsObj` | Sets columns. |
| `dtResponsiveOn` | Sets the responsive option so that the table will adapt for different screen sizes. |
| `dtResponsiveObj` | Turns on responsive behaviour and provides access to the responsive configuration options for DataTables. |
