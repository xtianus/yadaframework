# `net.yadaframework.web.datatables.config.YadaDataTableConfirmDialog`

| Method | Description |
|---|---|
| `dtPlaceholderColumnName` | Set the columns that hold the value for the placeholders |
| `dtTitle` | Set the title for the confirmation dialog |
| `dtMessageSingular` | Set the message for the confirmation dialog when one element is selected |
| `dtMessagePlural` | Set the message for the confirmation dialog when multiple elements are selected. |
| `dtConfirmButton` | Set the text for the confirmation button |
| `dtAbortButton` | Set the text for the cancel button |
