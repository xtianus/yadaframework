# `net.yadaframework.web.datatables.config.YadaDataTableConfirmDialog`

| Method | Description |
|---|---|
| `dtPlaceholderColumnName` | Sets the columns that hold the value for the placeholders |
| `dtTitle` | Sets the title for the confirmation dialog |
| `dtMessageSingular` | Sets the message for the confirmation dialog when one element is selected |
| `dtMessagePlural` | Sets the message for the confirmation dialog when multiple elements are selected. |
| `dtConfirmButton` | Sets the text for the confirmation button |
| `dtAbortButton` | Sets the text for the cancel button |
