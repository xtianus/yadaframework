# `net.yadaframework.components.YadaNotifyData`

| Method | Description |
|---|---|
| `add` | Appends the notification to the current response. |
| `keepLoader` | The notification modal does not turn off any existing loader |
| `message` | Set the notification message. |
| `autoclose` | Set the autoclose time in milliseconds - no close button is rendered |
| `center` | Vertically center the modal (with Bootstrap 4) |
| `addClasses` | Add a space-separated list of classes to add to the notification .modal-dialog element |
| `reloadOnClose` | Set the page to reload when the modal is closed |
| `messageKey` | Set the notification message using localization. |
| `redirectOnClose` | The page will redirect on modal close |
| `callScript` | Add a script id to call when opening the notification modal. |
| `ok` | Set the notification severity if active is true |
| `info` | Set the notification severity if active is true |
| `error` | Set the notification severity if active is true |
