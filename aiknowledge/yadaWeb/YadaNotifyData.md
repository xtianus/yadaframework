# `net.yadaframework.components.YadaNotifyData`

| Method | Description |
|---|---|
| `add` | Appends the notification to the current response. |
| `keepLoader` | The notification modal does not turn off any existing loader |
| `message` | Sets the notification message. |
| `autoclose` | Sets the autoclose time in milliseconds - no close button is rendered |
| `center` | Vertically center the modal (with Bootstrap 4) |
| `addClasses` | Adds a space-separated list of classes to add to the notification .modal-dialog element |
| `reloadOnClose` | Sets the page to reload when the modal is closed |
| `messageKey` | Sets the notification message using localization. |
| `redirectOnClose` | The page will redirect on modal close |
| `callScript` | Adds a script id to call when opening the notification modal. |
| `ok` | Sets the notification severity if active is true |
| `info` | Sets the notification severity if active is true |
| `error` | Sets the notification severity if active is true |
