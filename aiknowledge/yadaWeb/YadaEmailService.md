# `net.yadaframework.components.YadaEmailService`

| Method | Description |
|---|---|
| `buildLink` | Convert a site-relative link to absolute, because in emails we can't use @{. |
| `sendSupportRequest` | Sends the configured support-request email. |
| `sendHtmlEmail` | Send a HTML email. |
| `loadTemplate` | Loads a template. |
| `getTemplatePath` | Returns a template path. |
| `getMailTemplateFile` | Returns a mail template file. |
| `sendEmail` | Send an email by specifying the content directly, without a template |
| `sendEmailBatch` | Sends a batch of prepared emails. |
| `timestamp` | Returns a formatted timestamp string. |
