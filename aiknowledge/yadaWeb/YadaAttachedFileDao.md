# `net.yadaframework.persistence.repository.YadaAttachedFileDao`

| Method | Description |
|---|---|
| `purgeYadaAttachedFile` | Deletes from db and disk |
| `delete` | Deletes a YadaAttachedFile and connected data, but not the file on disk. |
| `swapSortOrder` | Swaps the sortOrder of two YadaAttachedFile entities |
| `save` | Saves the current data. |
| `addTo` | Adds a yadaAttachedFile to a list that could be lazy |
| `find` | Finds an attached file by ID. |
