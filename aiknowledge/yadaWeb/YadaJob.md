# `net.yadaframework.persistence.entity.YadaJob`

| Method | Description |
|---|---|
| `activate` | Sets the job state to active and return the previous state |
| `pause` | Sets the job state to paused and return the previous state |
| `disable` | Sets the job state to disabled and return the previous state |
| `complete` | Sets the job state to completed and return the previous state |
| `incrementErrorStreak` | Increment the error streak count returning the new value |
