# `net.yadaframework.components.YadaNotify`

| Method | Description |
|---|---|
| `empty` | Creates an empty notification response. |
| `title` | Initialise the instance with a title |
| `titleKey` | Initialise the instance with a title formed with a key and its optional parameters for the message.properties in the Request locale |
| `clearAll` | Clear all previously added messages. |
| `isNotificationPending` | Test if a modal is going to be opened when back to the view (usually after a redirect) |
| `isErrorSet` | Return true if error() has been called before |
