# `net.yadaframework.web.datatables.config.YadaDataTableColumn`

| Method | Description |
|---|---|
| `dtCssClasses` | css classes to set on the cell. |
| `dtOrderableOff` | Makes the column not orderable |
| `dtSearchableOff` | Makes the column not searchable |
| `dtName` | Sets a name on this column for database operations and cross reference. |
| `dtOrderAsc` | Choose this column for the initial sorting, ascending |
| `dtOrderDesc` | Choose this column for the initial sorting, descending |
| `dtResponsivePriority` | In a responsive table control the order in which columns are hidden. |
| `back` | Sets back. |
