# `net.yadaframework.web.datatables.config.YadaDataTableHTML`

| Method | Description |
|---|---|
| `dtColumnObj` | Add a new column to the table. |
| `dtButtonObj` | Add a new button to the table. |
| `dtFooter` | Shows a footer with the same content as the header |
| `dtColumnCheckbox` | Adds the leftmost column with select checkboxes |
| `dtColumnCommands` | Adds the rightmost column with command icons |
| `dtCssClasses` | Set the css classes to set on the table tag. |
| `back` | Sets back. |
