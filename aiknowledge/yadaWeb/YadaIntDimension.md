# `net.yadaframework.raw.YadaIntDimension`

| Method | Description |
|---|---|
| `biggestCover` | Returns the dimension with the width and height that cover all given dimensions. |
| `biggest` | Returns the dimension with the width or height that has the biggest value, or the first one if there are many that qualify. |
| `smallest` | Returns the smallest dimension, or the first one if there are many of the same size |
| `isBiggerThan` | Returns true if both dimensions are not smaller than the argument, nor both equal. |
| `isSmallerThan` | Returns true if both dimensions are not bigger than the argument, nor both equal. |
| `isAnyBiggerThan` | Returns true if at least one dimension is bigger. |
| `isAnySmallerThan` | Returns true if at least one dimension is smaller. |
| `isEqualTo` | Returns true if this instance is equal to the argument |
